:::caution
Dit commando is nu afgeschreven ten gunste van het `blitz prisma` commando. 
Het `blitz db seed` commando werkt nog totdat Prisma de seeding functie ondersteunt.
:::

### `blitz db seed` {#blitz-db-seed}

Dit zal het `db/seeds.ts` script uitvoeren. 
Dit wordt vaak gebruikt om snel de ontwikkelomgeving op te zetten. Voor voorbeelden zie [seeds docs](database-seeds)
