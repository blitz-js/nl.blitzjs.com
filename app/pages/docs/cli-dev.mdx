**Alias: `blitz d`**

Start de Blitz ontwikkelingsserver.

#### Opties

| Optie                   | Afkorting | Bescrijving                                            	| Standaard       |
| ------------------------ | --------- | ------------------------------------------------------ | ------------- |
| `--hostname`             | `-H`      | Stel de hostnaam in die voor de server moet worden gebruikt.               | `"localhost"` |
| `--port`                 | `-p`      | Stel de poort in waarop je de server wilt laten luisteren. | `3000`        |
| `--no-incremental-build` |           | Schakel incremental build uit en begin vanuit een nieuwe cache. | `false`       |

#### Voorbeelden

```bash
blitz dev
blitz dev -p 4000
```
