**Alias: `blitz s`**

Start de Blitz productieserver.

#### Opties

| Optie       | Afkorting | Beschrijving                                    | Standaard       |
| ------------ | --------- | ------------------------------------------------ | ------------- |
| `--hostname` | `-H`      | Stel de hostnaam in die voor de server moet worden gebruikt.         | `"localhost"` |
| `--port`     | `-p`      | Stel de poort in waarop je de server wilt laten luisteren.| `3000`        |

#### Voorbeelden

:::note
Zorg ervoor dat je eerst `blitz build` en dan `blitz start` uitvoert
:::

```bash
blitz start
blitz start -H 127.0.0.1 -p 5632
```
