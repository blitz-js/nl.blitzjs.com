**Alias: `blitz p`**

Dit laadt al je omgevingsvariabelen en stuurt dan alle argumenten door naar de Prisma CLI.

Hopelijk voegt Prisma volledige ondersteuning voor omgevingsvariabelen toe zodat we dit uiteindelijk kunnen verwijderen :)

#### Voorbeelden

```
blitz prisma generate
blitz prisma migrate dev
blitz prisma migrate deploy
```
