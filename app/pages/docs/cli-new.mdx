Genereert een nieuw Blitz project in de huidige map. Het project bevat een basisstructuur voor een landingspagina, 
een database en alle andere configuraties die de magie van Blitz laten werken.

```bash
blitz new [name]
```

| Argument | Vereist |                                   Beschrijving                                    |
| -------- | ------- | --------------------------------------------------------------------------------- |
| `name`   | Ja      | De naam van het project. Zal gebruikt worden voor de initiële project scaffolding |

#### Opties

|  Argument   | Afkorting |                                                                   Beschrijving                                                                   | Standaard |
| ----------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | --------- |
| `--npm`     |           | Gebruikt `npm` als de package manager voor het project. Indien weggelaten, zal het project standaard `yarn` gebruiken als deze is geïnstalleerd. | `false`   |
| `--dry-run` | `-d`      | Toont welke bestanden zouden worden gegenereerd, maar maakt de bestanden niet aan.                                                               | `false`   |
| `--no-git`  |           | Slaat het maken van een Git repository over.                                                                                                     | `false`   |

#### Voorbeelden

```bash
blitz new task-manager
```

```bash
blitz new task-manager --npm
```
