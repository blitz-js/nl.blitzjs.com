De Blitz CLI is je enige toegangspunt voor interactie met je app, van databasemanagement tot codegeneratie. 
De CLI is adviesgericht en begeleidt je naar best practices, maar is nog steeds flexibel genoeg zodat je jouw project kan bouwen zoals je dat wilt.

### Installatie {#installation}

Je hebt waarschijnlijk al gewerkt met de Blitz CLI bij het opzetten van je app. Wij raden je aan om Blitz globaal te installeren zodat je snel Blitz commando's op aanvraag kan uitvoeren:

```bash

# npm
npm i -g blitz

# Yarn
yarn global add blitz
```

### Beschikbare commando's {#available-commands}

In de volgende secties kan je meer leren over de verschillende CLI commando's.

Daarnaast kan je `blitz help` of `blitz [COMMAND] --help` uitvoeren om rechtstreeks in je terminal hulp te krijgen.
