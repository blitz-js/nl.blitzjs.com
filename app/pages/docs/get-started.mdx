### Installeer Blitz {#install-blitz}

Run `yarn global add blitz` or `npm install -g blitz --legacy-peer-deps`
(legacy-peer-deps is needed because npm totally change behavior of peer
deps and some dependencies haven't caught up)

### Maak een nieuwe applicatie {#create-a-new-app}

1. `blitz new myAppName`
2. `cd myAppName`
3. `blitz dev`
4. Bekijk je gloednieuwe applicatie op
   [http://localhost:3000](http://localhost:3000)
